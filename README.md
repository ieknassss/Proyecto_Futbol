# Sistema de Posiciones del Fútbol

## Versión de Python utilizada
Python 3.14.3

## Descripción del proyecto

Este proyecto consiste en un sistema informativo desarrollado en Python que permite consultar las principales posiciones del fútbol y sus funciones dentro del campo de juego.

El programa funciona mediante un menú interactivo en consola donde el usuario puede seleccionar una posición específica (Portero, Defensa, Mediocampista o Delantero) y visualizar información detallada sobre sus responsabilidades y características principales.

Para mejorar la presentación visual, se utilizó la librería externa "rich", la cual permite mostrar tablas formateadas y textos con estilo, ofreciendo una interfaz más profesional y organizada.

El proyecto fue desarrollado utilizando un entorno virtual para gestionar correctamente las dependencias y evitar conflictos con otros proyectos. Además, incluye el archivo requirements.txt para facilitar la instalación de las librerías necesarias en otro entorno.

Se implementó control de versiones mediante Git y GitHub, manteniendo un historial organizado de los cambios realizados.

## Instalación

1. Crear la carpeta y proyecto.
2. Crear el entorno virtual:

   python -m venv venv

3. Activar el entorno virtual:

   venv\Scripts\activate

4. Instalar las dependencias:

   pip install -r requirements.txt

## Ejecución

Para ejecutar el programa:

python primerprograma.py
